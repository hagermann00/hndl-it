# GEMINI IMAGE PRO: FULL-FORM PROMPT TEMPLATE

## The Master Prompt Structure

Use this template as a starting point, then iterate conversationally.

---

## TEMPLATE A: Photorealistic Scene

```
Create a photorealistic image of [SUBJECT DESCRIPTION] 
in [LOCATION/SETTING], 
during [TIME OF DAY/LIGHTING CONDITION].

Style: Professional photography, [CAMERA/LENS if known]
Mood: [EMOTIONAL TONE]
Colors: [COLOR PALETTE or "natural"]
Composition: [FRAMING - wide, medium, close-up]
Quality: High resolution, 4K, sharp detail

Additional details:
- [SPECIFIC ELEMENT 1]
- [SPECIFIC ELEMENT 2]
- [SPECIFIC ELEMENT 3]
```

### Example (Photorealistic)

```
Create a photorealistic image of a weathered wooden fishing boat 
resting on a misty Scandinavian shoreline at dawn.

Style: Professional landscape photography, 24mm wide-angle lens
Mood: Serene, contemplative, slightly melancholic
Colors: Muted blues, grays, with warm amber highlights from rising sun
Composition: Wide shot, boat in lower third, dramatic sky above
Quality: High resolution, 4K, sharp detail

Additional details:
- Gentle fog rolling across the water
- Rope coiled on the boat's deck
- Distant pine trees barely visible through the mist
```

## TEMPLATE B: Book Cover (KDP-Ready)

```
Create a professional book cover for "[BOOK TITLE]" by [AUTHOR NAME].

Genre: [GENRE]
Target audience: [AUDIENCE]
Orientation: Portrait, 1600x2560 pixels (KDP eBook ratio)

Visual concept:
[DESCRIBE THE MAIN VISUAL ELEMENT OR SCENE]

Typography requirements:
- Title: [FONT STYLE], [POSITION - top, center, etc.]
- Author name: [FONT STYLE], [POSITION]
- Optional tagline: "[TAGLINE TEXT]"

Style: [ARTISTIC STYLE]
Mood: [EMOTIONAL ATMOSPHERE]
Color palette: [PRIMARY COLORS]

Important: Leave 0.25 inch safe zone from edges for text placement.
```

### Example (Book Cover)

```
Create a professional book cover for "The Last Algorithm" by Sarah Chen.

Genre: Science Fiction / Thriller
Target audience: Adult readers of techno-thrillers
Orientation: Portrait, 1600x2560 pixels (KDP eBook ratio)

Visual concept:
A lone figure silhouetted against a massive holographic data visualization,
standing in a dark server room with blue and cyan lighting.

Typography requirements:
- Title: Bold sans-serif, positioned in upper third
- Author name: Clean sans-serif, bottom center
- Optional tagline: "When the code becomes conscious"

Style: Cinematic, modern, high-tech aesthetic
Mood: Tense, mysterious, cutting-edge
Color palette: Deep blacks, electric blues, cyan accents, subtle purple

Important: Leave 0.25 inch safe zone from edges for text placement.
```

## TEMPLATE C: Character Design

```
Design a character: [CHARACTER NAME/ROLE]

Physical description:
- Age: [AGE RANGE]
- Build: [BODY TYPE]
- Distinguishing features: [UNIQUE CHARACTERISTICS]
- Clothing/outfit: [DETAILED CLOTHING DESCRIPTION]
- Accessories: [ITEMS THEY CARRY/WEAR]

Expression: [FACIAL EXPRESSION/EMOTION]
Pose: [BODY POSITION]
Background: [SIMPLE/COMPLEX, DESCRIBE]

Art style: [REALISTIC/STYLIZED/ANIME/etc.]
Color palette: [DOMINANT COLORS]
Lighting: [LIGHTING SETUP]

Purpose: [CHARACTER SHEET/SINGLE ILLUSTRATION/BOOK INTERIOR]
```

### Example (Character Design)

```
Design a character: Captain Vera Okonkwo, space freighter pilot

Physical description:
- Age: Early 40s
- Build: Athletic, practical physique
- Distinguishing features: Silver streak in black hair, cybernetic left eye (subtle glow), small scar on chin
- Clothing/outfit: Worn leather flight jacket over utilitarian jumpsuit, magnetic boots
- Accessories: Holstered sidearm, wrist communicator, captain's insignia patch

Expression: Confident half-smile, eyes showing experience and wisdom
Pose: Standing with arms crossed, weight on one leg, relaxed authority
Background: Simple gradient, focus on character

Art style: Semi-realistic, graphic novel aesthetic
Color palette: Earth tones (brown jacket), muted greens and grays (jumpsuit), amber eye glow
Lighting: Three-quarter lighting from upper left, subtle rim light

Purpose: Character reference sheet for illustrated novel
```

## TEMPLATE D: Interior Illustration (KDP)

```
Create an interior illustration for a [GENRE] book.

Scene description:
[DETAILED DESCRIPTION OF THE MOMENT/SCENE TO ILLUSTRATE]

Style: [LINE ART/FULL COLOR/GRAYSCALE]
Dimensions: [WIDTH x HEIGHT in pixels]
Resolution: 300 DPI quality

Mood: [EMOTIONAL TONE OF THE SCENE]
Key elements that MUST appear:
- [ELEMENT 1]
- [ELEMENT 2]
- [ELEMENT 3]

Exclude: [ANYTHING THAT SHOULD NOT APPEAR]

Format notes: Clean edges, suitable for book printing, 
[WITH/WITHOUT] border
```

### Example (Interior Illustration)

```
Create an interior illustration for a fantasy adventure book.

Scene description:
A young wizard discovering an ancient library hidden behind a waterfall,
the first rays of sunlight streaming through the water creating 
prismatic light across towering bookshelves.

Style: Detailed line art with cross-hatching, black and white
Dimensions: 1200 x 1800 pixels
Resolution: 300 DPI quality

Mood: Wonder, discovery, magical revelation
Key elements that MUST appear:
- The young wizard in traveling cloak, back to viewer
- Cascading waterfall framing the entrance
- Ancient books and scrolls on curved shelves
- Dust motes visible in the light beams

Exclude: Any modern elements, other characters

Format notes: Clean edges, suitable for book printing, 
decorative border with Celtic knot pattern
```

## TEMPLATE E: Marketing/Social Media

```
Create a [PLATFORM] graphic for [BRAND/PURPOSE].

Dimensions: [EXACT DIMENSIONS for platform]
- Instagram Post: 1080x1080
- Instagram Story: 1080x1920
- Facebook Cover: 820x312
- YouTube Thumbnail: 1280x720

Content:
- Main visual: [DESCRIBE]
- Text overlay: "[EXACT TEXT]" in [FONT STYLE]
- Logo placement: [POSITION] (or "no logo")

Brand colors: [HEX CODES or descriptions]
Style: [MODERN/VINTAGE/PLAYFUL/PROFESSIONAL/etc.]
Call to action: [IF APPLICABLE]
```

### Example (Marketing)

```
Create an Instagram Post graphic for a coffee subscription service.

Dimensions: 1080x1080 (square)

Content:
- Main visual: Steaming cup of artisan coffee with latte art, 
  surrounded by scattered coffee beans on dark wood surface
- Text overlay: "WAKE UP DIFFERENT" in bold sans-serif, top center
- Logo placement: Bottom right corner, small

Brand colors: Deep brown (#3C2415), cream (#F5F0E6), gold accent (#D4AF37)
Style: Modern minimalist with warm, inviting feel
Call to action: Small text "Subscribe Now" below main text
```

## ITERATION COMMANDS

After initial generation, use these follow-up prompts:

| Goal | Command |
|------|---------|
| Adjust lighting | "Make the lighting [warmer/cooler/more dramatic/softer]" |
| Change colors | "Change the [element] color to [new color]" |
| Add element | "Add [object] in the [location]" |
| Remove element | "Remove the [object]" |
| Adjust composition | "Zoom in on [area]" or "Show more of the [area]" |
| Style shift | "Make this more [style descriptor]" |
| Text fix | "Change the text to read '[new text]'" |
| Resolution | "Generate at maximum resolution" |
